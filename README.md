- Character Arrays

Each problem is solved using separate functions to make the code simple, readable, and easy to understand.

---

## Problems Included

### Problem 1 – Average in an Array
- Store student marks in an array.
- Calculate the average using a function.
- Display the average up to two decimal places.

### Problem 2 – Largest and Smallest Values
- Store marks in an array.
- Find the highest score.
- Find the lowest score.
- Display both values.

### Problem 3 – Custom Power Function
- Calculate the power of a number.
- Do not use the built-in `pow()` function.
- Use a loop to calculate the result.

### Problem 4 – Voltage Calculator
- Calculate voltage using Ohm's Law.
- Formula:
  ```
  Voltage = Current × Resistance
  ```
- Display the calculated voltage.

### Problem 5 – Cricket Team Initial Finder
- Store player initials in a character array.
- Use Linear Search to find a player's initial.
- Display whether the player is found or not.

### Problem 6 – Word Length Calculator
- Input a word.
- Find its length without using `strlen()`.
- Count characters until the null character (`'\0'`) is reached.
